# RFC 8446 - The Transport Layer Security (TLS) Protocol Version 1.3

This document specifies version 1.3 of the Transport Layer Security (TLS) protocol. TLS allows client/server applications to communicate over the Internet in a way that is designed to prevent eavesdropping, tampering, and message forgery.

TLS 1.3 includes a downgrade protection mechanism embedded in the server's random value: a server negotiating TLS 1.2 or earlier sets the final eight bytes of its random value to a specific sentinel, which a client can detect and reject to prevent version-downgrade attacks.
